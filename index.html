Bienvenido a mi generador de plantillas

¿ Qué es Jekyll ?

Jekyll es un generador de sitios estáticos. Toma texto escrito en su lenguaje de marcado favorito y usa diseños para crear un sitio web estático. Puede modificar la apariencia del sitio, las URL, los datos que se muestran en la página y más.

🚨 Prerequisitos

Versión de Ruby 2.5.0 o superior, incluidos todos los encabezados de desarrollo (verifique su versión de Ruby usando ruby ​​-v) RubyGems (verifique su versión de Gems usando gem -v) GCC y Make (verifique las versiones usando gcc -v, g ++ -v y make -v)

🧮 Instrucciones

1. Instala las gemas de jekyll y bundler.

gem install jekyll bundler

2. Cree el sitio y hágalo disponible en un servidor local

* Nota 🚧: Antes de generar el sitio, diríjase al archivo "_config.yml" y cambie el atributo "baseurl".

- Si desea correr el proyecto localmente, iguale a ""

- Si desea correr el proyecto remotamente, iguale a "templategenerator"

bundle exec jekyll serve

3. Browse to http://localhost:4000

👷🏻‍♀️ Generación del directorio "docs"
`jekyll build` or `bundle exec jekyll build`
