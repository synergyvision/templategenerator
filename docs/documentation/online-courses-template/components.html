Cards
Blog-card.html

A blog card that displays the main information of the blog.

Parameters

url: The path to redirect

image-directory: The image source directory

image: Object wich include: name, the name of tha image. alt: Alternative text to image

title: The title of the blog post

description: Breve descriptionof the blog post

buy-card.html

This card includes the payment information, including prices, discount prices, coupons etc.

Parameters:

pay-info: Object wich includes: text, the title of the card, currency, the current currency, percentage-off-text, wich displays an "off", coupon-input wich include the parameters of an input, and a btn, wich includes the parameter of a button component

price: the current toal price

old-price: the total price without discount

svg-directory: source of the svg icon

category-card.html

This component shows a rounded button

Parameters:

name: Displays the title of the button

href: path to redirect

community-card.html

This component displays an element with a drop shape, and an image inside

Parameters:

member: Object that includes a name, a text (short text), and an image, wich includes a name (name of the image), and an alt (alternative to the image)

image-directory: The image directory

course-card.html

Parameters:

card: Object wich includes a name, owner, image (wich includes a name, and an alt), score (number between 0 and 5), discount (can be true or false), currency, discount-price (if discount equal to true), and an url to redirect

bought-info: Displays the progress of the course. Optional. Object wich includes completed (the percentage of completion), rating ( number between 0 and 5)

added: Set to true if you want to display the added buttons

course-feature-card.html

This component displays a card with listed texts

Parameters:

title: Displays the name of the card

section: Array that includes, an icon, and a text

svg-directory: The path of the svg icons

course-membered-questions.html

This components displays a card a question card, including the person image, the question and the description.

Parameters:

question: Object that includes a question (the question by it self), an answer, and a person with an image, wich includes a name and an alt (alternative text to image)

credit-card-edit.html

This component display a card with the credit card information.

Parameters:

This component has no parameters, so the information is taken from the data directory

instructor-card.html

This component displays a breve profile of an instructor.

Parameters:

title: displays the title of the card

person: Object that includes a name, and image (wich includes a name, and an alt), a short-description of the instructor, features (an array wich includes an icon and a text), and a href wich redirect to a page

image-directory: directory of the instructor image

btn-more: Object that includes the button component parameters

price-card.html

This component displays a buy card for a course, showing the prices and a list of buttons.

Parameters:

price: The price of the course

discount-price: The price after the discount

buttons: An array wich includes an href, and an icon

svg-directory: the directory of the svg icons

image-directory: the directory of the image
