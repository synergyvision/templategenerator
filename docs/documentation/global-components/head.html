Head
Description

This component includes the head of the html structure. It is usually added in the default layout of the template, and receives a number of parameters that allow to display the information of the pages in a different way.

How to use
{% include global/head.html site=my-site-object script=my-particular-scripts %}
Parameters
  • site: Objeto que incluye:

    • title: nombre del sitio. Se mostrará si la pagina actual no tiene titulo.

    description: descripcion del sitio. Se mostrará si la pagina actual no posee descripcion.

    creator: Cuenta del creador

    twitter: Cuenta de twitter del creador

    css: dirección del main css

    logo-social-media: Logo a mostrar en las redes sociales. Se mostrará si la pagina actual no posee imagen.

    favicon: favicon del sitio.

    description: descripcion del sitio. Se mostrará si la pagina actual no posee descripcion.

    base-url: directorio base del sitio

    directories. objeto que incluye los directorios

  • script: Si desea incluir mas etiquetas 'script', añadalas a un componente, y pase la direccion de dicho componente aquí.
