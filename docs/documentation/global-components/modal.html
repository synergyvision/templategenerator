Modal
Description

This component consists of a popup with an overlay, which allows you to include any component within it

How to use
{% include global/modal.html title=your-modal-title id=your-modal-id file=your-component-path %}
Parameters
  • title: Title of the respective modal
  • id: id of the modal window. Make reference with a 'href = your-modal-id' in the button that will activate this modal
  • file: Address of the component that you want to include within the modal
