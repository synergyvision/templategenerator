Carousel
Description

This component allows us to add within it a number of specific cards, and show it in different slices.

How to use
{ % include global/carousel.html id=1 cardsFile='your-component-direction.html' quantity='3.0' cardsCollection=your-collection color=your-color % }
Parameters
  • cardsFile: Defines the path of the component that you want to include within the carousel.
  • quantiy: Defines the amount of components for each face of the carousel.
  • cardsCollection: Describe the collection that you want to show in the carrousel
  • color: Defines the color of the carrousel buttons.
