Progress bar
Description

This component animatedly displays a progress bar given a percentage.

How to use
{% include global/progress-bar.html percentage='30%' color=my-color name=my-bar-name %}
Parameters
  • color: Renders the progress bar of this color.
  • name: Name of the progress bar
  • percentage: Percentage to be rendered
